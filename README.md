# Examen Ley de Transito

Pagina estatica en HTML, CSS y JavaScript para practicar un examen de seleccion unica con 150 preguntas.

## Uso local

Abre `index.html` en el navegador. El progreso se guarda automaticamente con `localStorage` en el mismo navegador.

## Publicar en GitHub Pages

1. Sube estos archivos al repositorio.
2. En GitHub entra a `Settings > Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda la configuracion.

La pagina quedara disponible en la URL que GitHub Pages indique.
